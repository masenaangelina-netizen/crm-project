# Decathlon Taiwan: Q2 Awaken Customer Product Preference Analysis

## Overview
This repository contains the BigQuery SQL script used to analyze the purchasing behavior and product preferences of **"Awaken" (reactivated) members** for Decathlon Taiwan. The goal of this query is to determine which specific departments, families, and categories are driving the most Gross Merchandise Value (GMV) and volume among customers who have recently returned to shop with us.

## Business Definitions
To ensure data accuracy, this query relies on the following standardized business definitions:
* **Awaken Customer:** A member whose `member_active_category` is currently flagged as `'awaken'` in the member database. 
* **Timeframe:** between **April 1, 2026** and **June 30, 2026**.
* **Metrics Tracked:** The query aggregates the total number of unique buyers, total GMV (Gross Merchandise Value), and total items sold per product hierarchy.

## Data Sources (Tables Used)
This query pulls from the `dashboard_data_new` dataset within the `twdatawarehouse-bsv8` BigQuery project:
* **`tw_sales_detail_new`**: Contains granular transactional data, including recorded dates, GMV amounts, quantities, and product hierarchies (Department, Family, Category).
* **`tw_member_new`**: Contains member metadata, specifically the current active category status of the customer.

## Expected Output
When executed in BigQuery, the resulting dataset is sorted by highest Total GMV and outputs the following columns:

| Column Name | Data Type | Description |
| :--- | :--- | :--- |
| `department` | String | The top-level product grouping (e.g., Mountain, Water Sports). |
| `family` | String | The secondary product grouping. |
| `category` | String | The granular product category. |
| `unique_customers` | Integer | The distinct count of 'awaken' members who purchased from this category. |
| `total_gmv` | Numeric | The total Gross Merchandise Value generated by these customers. |
| `total_items` | Integer | The total quantity of physical items purchased. |

## How to Run This Query
1. Open the [Google Cloud BigQuery Console](https://console.cloud.google.com/bigquery).
2. Ensure you have access to the `twdatawarehouse-bsv8` project.
3. Open the SQL file associated with this README.
4. Copy the SQL code and paste it into the BigQuery editor.
5. Click **Run**.
